About This System

This face recognition attendance system uses MobileNetV2 embeddings and cosine similarity matching for accurate identification. Each registered student's face is converted to a unique 1280-dimensional vector that enables fast and reliable recognition.

Fast Recognition

Instant face matching with no retraining required. Students can be registered and recognized immediately.

Privacy Protected

Student photos are stored securely and not publicly accessible. Only authorized users can access sensitive data.

Cloud Ready

Optimized for deployment on platforms like Hugging Face Spaces with persistent storage support.
